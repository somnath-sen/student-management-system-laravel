**EdFlow** is an enterprise-grade, cloud-based Smart Campus Management System developed as an MCA Final Semester Project. It streamlines university operations, bridges the communication gap between administration, teachers, and students, and introduces modern capabilities like AI assistance and real-time GPS tracking.

---

## ✨ Key Features

* **🔐 Multi-Role Architecture:** Secure, distinct dashboard portals for Administrators, Teachers, and Students with strict Role-Based Access Control (RBAC).
* **🤖 StudyAI Integration:** A next-generation conversational AI assistant embedded directly into the student portal to help with academic queries.
* **📍 Live Family Tracker:** Real-time GPS location pinging and interactive maps to ensure student safety on and off-campus.
* **🪪 Smart Digital ID:** Auto-generated, mobile-ready digital identity cards featuring scannable QR codes for instant verification.
* **📄 Advanced Academic Engine:** Dynamic PDF marksheet generation and a highly secure, official Admit Card publication workflow.
* **📅 Dynamic Timetables:** An interactive "Kanban-style" class routine builder for Admins, which automatically generates personalized schedules for Teachers and Students.
* **💳 Financial Management:** Transparent fee tracking, due alerts, and transaction history.
* **✨ Premium UI/UX:** Built with Tailwind CSS featuring glassmorphism, animated page loaders, dark mode support, and a 100% mobile-responsive design.

---

## 🛠️ Technology Stack

* **Backend:** Laravel 11.x (PHP 8.2+)
* **Frontend:** Blade Templates, Tailwind CSS, Alpine.js, HTML5/CSS3
* **Database:** MySQL
* **Icons & Typography:** FontAwesome 6, Plus Jakarta Sans
* **Third-Party APIs:** Google Gemini API (StudyAI), QR Server API, Geolocation API
